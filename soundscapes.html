Soundscapes
Creative coding, Web Design
Soundscapes is a web-based tool that allows the user to use hand motions to create interesting textured audio scapes with over 700,000 sounds from the Freesound API. The piece was featured as a performance at the PARSONS X IMPACT: SEAPORT SATELLITE | NYCxDesign Exhibition in May 2026. Visit here!
Tools:
HTML/CSS/Javascript, Claude AI, Google MediaPipe, Freesound API
The tool uses vertical motion to control pitch and radius of the circle (based on spread of fingers or distance from camera) to control volume. The mechanics are inspired by a theremin, or hand-controlled audience choirs like Jacob Collier's famous concert act. Each hand controls a separate sound. You can record, layer and download you audioscape.
